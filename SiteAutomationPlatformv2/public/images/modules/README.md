# Module Images Directory

This directory contains images for the GTB modules (Page 7 - Wiring Diagram).

## Required Image Files:

### Add these 3 images to this folder:

1. **`ps.png`** or **`ps.jpg`**
   - Module: Power Supply (PS)
   - Slot 1 - Alimentation
   - Recommended size: 150x150px to 300x300px
   - Format: PNG (transparent background) or JPG

2. **`as-p.png`** or **`as-p.jpg`**
   - Module: AS-P Communication Port
   - Slot 2 - Port de Communication
   - Recommended size: 150x150px to 300x300px
   - Format: PNG (transparent background) or JPG

3. **`do12.png`** or **`do12.jpg`**
   - Module: DO12 Digital Outputs
   - Slot 3 - 12 Sorties Numériques
   - Recommended size: 150x150px to 300x300px
   - Format: PNG (transparent background) or JPG

## File Naming Rules:
- Use lowercase letters
- Use hyphens (-) for multi-word names
- Supported formats: `.png`, `.jpg`, `.jpeg`
- No spaces in filenames

## Example File Structure:
```
public/
  images/
    modules/
      ps.png          ← Power Supply image
      as-p.png        ← AS-P Communication Port image
      do12.png        ← DO12 Digital Outputs image
      README.md       ← This file
```

## Image Recommendations:
- Clear, professional technical drawings
- White or transparent background
- Square aspect ratio preferred
- High contrast for visibility
- SVG format is also supported (as `ps.svg`, `as-p.svg`, `do12.svg`)

## After Adding Images:
The images will automatically appear in the Slot cards on Page 7 (Wiring Diagram).
